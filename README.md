## Usage

`define <word>`

### Flags

- `-s`, `--synonyms` Show synonyms for definitions.
- `-a`, `--antonyms` Show antonyms for definitions.
- `-e`, `--examples` Show examples for definitions.
- `-f`, `--full` Show synonyms, antonyms, and examples.
